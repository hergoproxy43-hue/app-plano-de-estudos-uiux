# 📚 App Plano de Estudos

Projeto de UI/UX de um aplicativo mobile para ajudar estudantes a organizarem seus estudos.

---

## 🖼️ Preview do Projeto

![Preview do App](Plano%20de%20estudos.png)

---

## 🎯 Objetivo

O objetivo deste projeto é permitir que o usuário:

- Criar um plano de estudos personalizado
- Organizar horários
- Melhorar sua produtividade

---

## 🧠 Processo de Design

Este projeto inclui:

- Wireframe
- Estrutura da jornada do usuário
- Protótipo visual final
- Tela de confirmação

---

## 🎨 Ferramenta utilizada
